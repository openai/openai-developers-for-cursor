# Security Policy

## Reporting Security Issues

If you believe you have found a security vulnerability in this plugin, please
report it through OpenAI's [coordinated vulnerability disclosure policy](https://openai.com/policies/coordinated-vulnerability-disclosure-policy).

Please contact disclosure@openai.com for questions or concerns regarding the
security of OpenAI products or services.

## Responsible Disclosure

We appreciate the efforts of security researchers and individuals who help us
maintain the security of OpenAI software. If you believe you have found a
security vulnerability, please allow us a reasonable amount of time to
investigate and address the issue before making any information public.
